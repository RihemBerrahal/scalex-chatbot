### ğŸ§  Backend Setup (Node.js)
1. Navigate to the server folder  
   ```bash

   ```
2. Install dependencies  
   ```bash
   npm install
   ```
3. Create a `.env` file
   ```bash

   HF_TOKEN=your_huggingface_token

4. Run locally

   ```
5. Your backend should be available at  
   ```
   http://localhost:5050
   ```
